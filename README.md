jQuery.positionSticky
=====================
[![Code Climate](https://codeclimate.com/github/katranci/jQuery.positionSticky.png)](https://codeclimate.com/github/katranci/jQuery.positionSticky)

Usage
-----

```
 -------------------------- => window 
|   --------------------   |
|  |              ---   |==|=> container
|  |             |   |  |  |
|  |             |   |==|==|=> sticky  
|  |              ---   |  |
|  |                    |  |
|  |                    |  |
|  |                    |  |
|  |                    |  |
|   --------------------   |
|   --------------------   |
|  |                    |  |
|  |                    |  |

```

```javascript
$('#sticky').positionSticky();  
```
